# HR Bias Detection in AI-Based Hiring

This project demonstrates how **bias can appear in AI-based hiring systems** and how data analysis can be used to detect unfair hiring patterns.

The project uses a **synthetic candidate dataset** and Python-based analysis to evaluate whether hiring outcomes differ across groups such as **gender or educational background**.

The analysis is implemented using **Google Colab and Python libraries**.

---

## Project Objective

The objective of this project is to:

- Understand how **bias can occur in AI-assisted hiring systems**
- Analyze hiring outcomes using **data analytics**
- Detect potential **unfair patterns in selection decisions**
- Demonstrate **ethical AI practices in HR analytics**

---

## Dataset

A synthetic dataset of **100 candidates** is generated with attributes such as:

- Candidate ID
- Gender
- College Tier
- Years of Experience
- Interview Score
- Hiring Decision (Selected / Not Selected)

The dataset is generated using Python to simulate a realistic hiring scenario.
